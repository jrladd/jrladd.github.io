There is more than one DH.

DH can be understood as

  • a field
  • a set of methodologies
  • a community of practice

What’s under the DH umbrella?

  • digital pedagogy/classroom technologies
  • computational research/humanities data
  • web archives/digital editions/online resources
  • critical responses to technology (STS, critical code studies)

There is more than one job market.

Ac / Alt-Ac / Non-Ac

Academic jobs

  • in a primary field with DH as secondary interest
  • with DH/data/computation as primary field

Alt-Ac jobs

  • in DH centers (teachers, developers, program managers)
  • at university libraries
  • in archives/museums/cultural institutions
  • in professional organizations
